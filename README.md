# Robot Vacuum Cleaner Setup Test Design

This repository contains my submission for the Software Test Design task for the initial setup flow of a Robot Vacuum Cleaner (Unboxing to First Cleaning Run).

## Deliverable
- `Software_Test_Design_Robot_Vacuum_Cleaner_Seonghun.xlsx`

## Contents
The workbook includes:
- High-level test scenarios
- Detailed and directly automatable test cases
- At least 3 complex edge cases
- Top 5 prioritized test cases with rationale
- Automation strategy and hardest-to-automate area

## Notes
- All test cases are written in English.
- The scope covers the setup journey from app installation / onboarding through first cleaning run.
